# EasyAgentCenter

[English](README.md) | [简体中文](README.zh-CN.md) | [Русский](README.ru.md) | [日本語](README.ja.md) | [한국어](README.ko.md) | [Español](README.es.md)

EasyAgentCenter — это быстрый и легкий Windows‑менеджер для CLI Agent. Он помогает запускать и управлять Codex CLI, Kimi Code, Claude Code, Hermes и другими популярными агентами, а также любыми пользовательскими командами или инструментами, которые можно запустить из терминала.

Проект полезен тем, кто одновременно работает с несколькими программными агентами: можно просматривать сеансы по проектам или по Agent, общаться с ними во встроенном терминале, быстро запускать нужные рабочие каталоги и получать системные уведомления, когда сеанс завершился или завершился ошибкой. EasyAgentCenter не содержит API Key и не отправляет ваши данные сеансов. Каждый Agent использует собственное состояние входа и локальную конфигурацию.

## Возможности

- Автоматическое обнаружение установленных CLI Agent из PATH.
- Добавление папок проектов и быстрый запуск Agent в выбранном проекте.
- Управление сеансами по проектам или по Agent.
- Остановка, удаление, массовое удаление, сортировка и восстановление записей сеансов.
- Встроенный PTY‑терминал для прямого общения с Agent.
- Настраиваемый цвет фона терминала.
- Дополнительная панель квот Codex CLI для вывода `/status` и `/usage`.
- Подсказки команд установки и обновления для известных Agent.
- Необязательные уведомления рабочего стола после завершения или ошибки сеанса.
- Языки интерфейса: английский, упрощенный китайский, русский, японский, корейский и испанский.

## Конфиденциальность

EasyAgentCenter хранит метаданные сеансов и журналы сеансов локально на вашем компьютере. Само приложение не отправляет ваши данные сеансов и не содержит API Key. Отдельные CLI Agent могут использовать собственное состояние входа, переменные окружения или локальные настройки.

В разработческой среде локальные папки `data/` и `logs/` игнорируются Git и не должны попадать в репозиторий.

## Быстрый старт

### Запуск для разработки

```bash
npm ci
npm run dev
```

### Быстрый запуск одним файлом

Дважды щелкните:

```text
start-easy-agent-center.bat
```

Для скрытого запуска используйте:

```text
start-easy-agent-center-hidden.vbs
```

Скрытый запускатель стартует `npm run dev` без видимого окна Command Prompt. Для отладки проблем запуска лучше использовать `.bat` файл.

### Сборка

```bash
npm run dist:dir
```

Затем откройте распакованное приложение:

```text
dist\win-unpacked\easy-agent-center.exe
```

Портативный exe:

```bash
npm run dist
```

## Требования

- Node.js 24.14.0, см. `.nvmrc` / `.node-version`
- Основная целевая платформа — Windows
