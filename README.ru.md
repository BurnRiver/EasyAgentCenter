# EasyAgentCenter

[English](README.md) | [简体中文](README.zh-CN.md) | [Русский](README.ru.md) | [日本語](README.ja.md) | [한국어](README.ko.md) | [Español](README.es.md)

EasyAgentCenter — легкий Windows‑менеджер для CLI coding agents.

EasyAgentCenter объединяет Codex CLI, Claude Code, Kimi Code, Hermes и пользовательские CLI Agent в одном окне: сеансы по проектам, встроенный PTY‑терминал, уведомления рабочего стола и экспорт в Markdown.

Проект полезен тем, кто одновременно работает с несколькими программными агентами: можно просматривать сеансы по проектам или по Agent, общаться с ними во встроенном терминале, быстро запускать нужные рабочие каталоги, а также перезапускать или экспортировать сеансы из контекстного меню. EasyAgentCenter не содержит API Key и не отправляет ваши данные сеансов. Каждый Agent использует собственное состояние входа и локальную конфигурацию.

## Возможности

- Автоматическое обнаружение установленных CLI Agent из PATH.
- Добавление папок проектов и быстрый запуск Agent в выбранном проекте.
- Управление сеансами по проектам или по Agent.
- Перезапуск, остановка, удаление, массовое удаление, сортировка и восстановление записей сеансов.
- Встроенный PTY‑терминал для прямого общения с Agent.
- Экспорт журналов сеансов в Markdown.
- Настраиваемый цвет фона терминала.
- Дополнительная панель квот Codex CLI для вывода `/status` и `/usage`.
- Подсказки команд установки и обновления для известных Agent.
- Необязательные уведомления рабочего стола после завершения или ошибки сеанса.
- Языки интерфейса: английский, упрощенный китайский, русский, японский, корейский и испанский.

## Конфиденциальность

EasyAgentCenter хранит метаданные сеансов и журналы сеансов локально на вашем компьютере. Само приложение не отправляет ваши данные сеансов и не содержит API Key. Отдельные CLI Agent могут использовать собственное состояние входа, переменные окружения или локальные настройки.

Если вы запускаете проект из исходного кода, созданные папки `data/` и `logs/` используются только для локальной разработки. Они игнорируются Git и не должны попадать в репозиторий.

## Быстрый старт

### Запуск для разработки

```bash
npm ci
npm run dev
```

### Быстрый запуск одним файлом

Дважды щелкните:

```text
start-easy-agent-center.bat
```

Для скрытого запуска используйте:

```text
start-easy-agent-center-hidden.vbs
```

Скрытый запускатель стартует `npm run dev` без видимого окна Command Prompt. Для отладки проблем запуска лучше использовать `.bat` файл.

### Сборка

```bash
npm run dist:dir
```

Затем откройте распакованное приложение:

```text
dist\win-unpacked\easy-agent-center.exe
```

Портативный exe:

```bash
npm run dist
```

## Требования

- Обычным пользователям достаточно скачать portable exe; Node.js не требуется.
- Для разработки из исходного кода используется Node.js 24.14.0, см. `.nvmrc` / `.node-version`
- Основная целевая платформа — Windows
