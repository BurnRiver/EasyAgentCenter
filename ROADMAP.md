# EasyAgentCenter Roadmap

## MVP - Stable Session Console

- Keep agent discovery fast and readable.
- Keep project folders visible even when they have no sessions.
- Quick-start agents inside the last used or selected project.
- Make session stop/delete/reorder predictable.
- Make terminal input, paste, selection, and resize feel close to native `cmd` / PowerShell.
- Persist session metadata and transcript logs.

## Near Term

- Improve terminal rendering and IME behavior on Windows.
- Add session search and transcript export.
- Add per-agent launch profiles for common flags and environment variables.
- Keep Codex quota parsing optional and isolated.
- Improve install/update helper safety text.

## Deferred

- External messaging integrations.
- Cross-platform packaging.

These features can return later as separate modules, but they are intentionally outside the core EasyAgentCenter scope.
